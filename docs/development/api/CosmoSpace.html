CosmoSpace#

class cosmolayer.CosmoSpace(*args, **kwargs)[source]#

Implicit COSMOspace layer.

Solves the following implicit equation for the activity coefficient vector \(\boldsymbol{\gamma}\), given the segment-type weight vector \(\mathbf{x}\) (nonnegative, not necessarily normalized) and the reduced interaction energy matrix \(\hat{\mathbf{U}}_T = {(RT)}^{-1}\mathbf{U}\):

\[\boldsymbol{\gamma} \odot (\mathbf{B} (\mathbf{x} \odot \boldsymbol{\gamma})) = s \mathbf{1},\]

where \(\mathbf{B} = \exp(\hat{\mathbf{U}}_T)\) is the matrix of Boltzmann factors and \(s = \mathbf{1} \cdot \mathbf{x}\) is the sum of the segment-type weights. For a physically meaningful solution, \(s\) must be equal to 1.

Domain constraint: \(\mathbf{x} \geq \mathbf{0}\) elementwise, with at least one strictly positive component.

With \(s = 1\), the solution satisfies \(\min(\boldsymbol{\gamma}) > 0\) and \(\mathbf{a}^T \mathbf{B} \mathbf{a} = 1\), where \(\mathbf{a} = \mathbf{x} \odot \boldsymbol{\gamma}\) is the activity vector.

Even though \(\hat{\mathbf{U}}_T\) is usually symmetric, it is not assumed to be so.

Note

Supports batching, meaning that \(\mathbf{x}\) and \(\hat{\mathbf{U}}_T\) can have broadcastable leading dimensions, and all computations are vectorized along these dimensions.

Parameters:

  • x (torch.Tensor) – Segment-type distribution vector. Shape: (…, num_segment_types).

  • U_RT (torch.Tensor) – Reduced interaction energy matrix \(\hat{\mathbf{U}}_T = {(RT)}^{-1}\mathbf{U}\). Shape: (…, num_segment_types, num_segment_types).

  • max_iter (int) – Maximum number of iterations.

Returns:

gamma – The segment activity coefficient vector. Shape: (…, num_segment_types).

Return type:

torch.Tensor

Raises:

RuntimeError – If the fixed-point solver does not converge within max_iter iterations.

Examples

>>> import numpy as np
>>> from cosmolayer.cosmosac import Component, CosmoSac2002Model
>>> from importlib.resources import files
>>> components = [
...     Component.from_file(files("cosmolayer.data") / f"{species}.cosmo")
...     for species in ("C=C(N)O", "NCCO")
... ]
>>> probabilities = [
...     component.get_probabilities(merge=True)
...     for component in components
... ]
>>> x = torch.stack(
...     [torch.tensor(p, dtype=torch.float32) for p in probabilities],
... ).requires_grad_(True)
>>> U_RT = torch.tensor(
...     CosmoSac2002Model.create_interaction_matrices(298.15)[0],
...     dtype=torch.float32,
...     requires_grad=True,
... )
>>> gamma = CosmoSpace.apply(x, U_RT)
>>> gamma.log()
tensor([[ -5.2...,  -4.6...,  ... -13.3..., -14.5...],
        [-22.4..., -20.7...,  ... -4.8...,  -5.5...]], grad_fn=<LogBackward0>)
>>> loss = (gamma ** 2).sum()
>>> loss.backward()
>>> x.grad
tensor([[ 2.8...e+03,  6.0...e+02, ... -4.8...e+04],
        [-5.1...e+02, -5.0...e+02, ...  6.3...e+02]])
